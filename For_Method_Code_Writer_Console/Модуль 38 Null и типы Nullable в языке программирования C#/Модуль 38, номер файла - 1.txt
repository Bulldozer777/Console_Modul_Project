Одно из отличий ссылочных типов от типов значений состоит в том, 
что переменные ссылочных типов могут принимать значение null. Например:

object o = null;
string s = null;

Если переменным ссылочного типа не присваивается значение, 
то им дается значение по умолчанию - значение null. 
Фактически оно говорит об отстутсвии значения как такового.

Но типы значений, например, int, decimal, double и т.д. 
не могут принимать значение null.

Оператор ??
Оператор ?? называется оператором null-объединения. 
Он применяется для установки значений по умолчанию для типов, 
которые допускают значение null. Оператор ?? возвращает левый операнд,
 если этот операнд не равен null. Иначе возвращается правый операнд. 
При этом левый операнд должен принимать null.
